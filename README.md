# Pinecone Blog

Experimentation with the pinecone database

Instructions:

 1 - Create the poetry environment using the .toml file

 2 - Get an account on pinecone.io
 
 3 - Create a .env file 
 
 4 - Edit the .env file to include:
     
     API_KEY=<your api key>
 
 5 - Run main.py using: 
 
     poetry run python main.py
